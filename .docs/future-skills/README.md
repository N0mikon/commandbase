All three domains (code, vault, services) share the same RDSPI phase structure with domain-specific tooling at each step. `/naming-session` ties into phase-by-phase work across all domains. Brainstorming skills are the pre-RDSPI entry point for each domain. `/researching-repo` completes the research stack alongside `/researching-web`, `/researching-frameworks`, and `/researching-code`.
