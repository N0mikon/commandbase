# commandbase-meta

Skill, agent, and hook authoring tools for building Claude Code extensions. You don't need any other commandbase plugins for this one.

## Dependencies

None.

## Skills

| Skill | Description |
|-------|-------------|
| /auditing-agents | Audit existing agents for validation issues and fix compliance problems |
| /auditing-skills | Audit existing skills for validation issues and fix compliance problems |
| /creating-agents | Create new Claude Code agents from scratch or edit existing ones |
| /creating-hooks | Create Claude Code hooks, configure deny rules, debug hook execution issues |
| /creating-posts | Draft social media posts with anti-AI voice compliance across 5 platforms |
| /creating-skills | Create new Claude Code skills from scratch or convert agents into skills |

## Installation

```shell
/plugin install commandbase-meta
```
