## Skills

| Skill | Description |
|-------|-------------|
| /brainstorming-services | Explore direction for homelab or Docker service infrastructure before deployment |
| /designing-services | Make infrastructure architecture decisions: stack topology, networking, auth, monitoring |
| /implementing-services | Execute homelab service implementation plans from .docs/plans/ |
| /planning-services | Create phased deployment plans with success criteria and rollback steps |
| /researching-services | Research current homelab infrastructure: services, ports, volumes, networks, dependencies |
| /structuring-services | Map compose file layout, .env templates, proxy routes, and volume organization |

## Installation

