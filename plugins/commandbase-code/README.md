## Skills

| Skill | Description |
|-------|-------------|
| /brainstorming-code | Explore direction and preferences for a code feature before research or planning |
| /debugging-code | Debug issues using systematic investigation with hypothesis tracking |
| /designing-code | Make architectural decisions for a feature or system |
| /implementing-plans | Implement technical plans from .docs/plans/ with automated verification |
| /planning-code | Create or iterate on implementation plans with thorough codebase research |
| /researching-code | Research a codebase to understand how it works |
| /starting-refactors | Initialize brownfield refactors, establish scope, snapshot baseline state |
| /structuring-code | Map file placement, module organization, and dependency structure |

## Agents

| Agent | Description |
|-------|-------------|
| code-analyzer | Analyzes codebase implementation details for specific components |
| code-librarian | Finds similar implementations, usage examples, or existing patterns to model after |
| code-locator | Locates files, directories, and components relevant to a feature or task |
